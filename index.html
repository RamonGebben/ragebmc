RaGeBMC

RaGeBMC is a automated configuration voor Raspbmc. This configuration will allow you to add torrent files to transmission web interface. It will also download this rss feed from ShowRSS. Here you can make you're own account and add the feed url to .flexget/settings.json

RamonGebben maintains RaGeBMC
